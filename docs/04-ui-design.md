## 组件树

```
App
├── SearchPanel
│   ├── SearchInput          — 搜索框，模糊匹配线路名
│   └── RouteDropdown        — 下拉选择列表
├── MapView
│   ├── RoutePolyline        — 线路路径（Leaflet Polyline）
│   ├── StopMarkers          — 站点标记（小圆点 CircleMarker）
│   └── BusMarkers           — 公交车位置标记（带方向箭头）
└── InfoPanel
    ├── DirectionToggle      — 方向切换按钮
    ├── BusStatus            — 在线车辆数 + 刷新倒计时
    └── StopList             — 有序站点列表
```

## 交互流程

1. **打开页面** → 加载 `routes.json`，展示空地图（聚焦槟城 5.37°N, 100.37°E）
2. **输入搜索** → 模糊匹配线路名，展示下拉列表
3. **选择线路** → 加载对应 `route-details/{id}_{dir}.json`，在地图上绘制：
   - 线路路径（彩色 Polyline）
   - 站点标记（圆点 + hover 显示名称）
   - 地图自动 fitBounds 到线路范围
4. **开始轮询实时位置** → 每 15 秒请求一次 GTFS-RT，过滤当前线路车辆
5. **切换方向** → 重新加载对应方向的线路详情，保持实时轮询
6. **点击站点** → Popup 显示站点名称
7. **点击公交车** → Popup 显示车牌号、速度、更新时间

## 地图样式

- **线路路径**: 蓝色半透明线条 (weight: 5, opacity: 0.7)
- **站点标记**: 白色圆点，蓝色边框 (radius: 5)
- **公交车图标**: 自定义 SVG 图标，带方向指示（根据 bearing 旋转）
  - 移动中: 绿色
  - 静止（speed=0）: 灰色
